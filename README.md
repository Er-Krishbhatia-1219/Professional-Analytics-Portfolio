# Professional Analytics Portfolio

A modern AI-assisted professional portfolio website created to showcase my academic background, technical skills, analytics projects, certifications, resume, and professional profile in an organized and interactive format for recruiters and hiring teams.

---

## Features

- Professional profile overview  
- Education & technical skills section  
- Analytics project showcase  
- Resume integration  
- Contact & professional platform links  
- Responsive modern dashboard UI

## Tech Stack

- JavaScript

## Purpose

This portfolio was developed as a professional digital profile to present my educational journey, technical exposure, and analytics learning experience in a clean and accessible manner for recruitment purposes.

This portfolio was designed and structured with the assistance of AI tools to improve presentation, UI organization, and productivity.
